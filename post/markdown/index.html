Markdown常用语法

2026-06-18 · 笔记 · 2 分钟阅读 · 951 字

适合：日常笔记、背诵、快速查阅、通用所有MD编辑器

1. 标题（1-6级）

语法：# 数量代表标题层级

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

2. 基础文本样式

**加粗**
*斜体*
***加粗斜体***
~~删除线~~
==高亮==
`行内代码`
<u>下划线</u>

3. 换行 & 分割线

  • 分段：空一行

  • 强制换行：行尾两个空格+回车

  • 分割线：--- / ***

4. 列表

无序列表

- 项目1
- 项目2
  - 子项目

有序列表

1. 第一条
2. 第二条
   1. 子条目

任务清单

- [ ] 未完成
- [x] 已完成

5. 引用

> 一级引用
>> 嵌套二级引用

6. 代码块

行内代码

  `代码内容`

多行带高亮代码

  ```python
  # 指定语言：python / java / sql / bash
  print("hello")
  ```

7. 链接、图片

超链接

[显示文字](链接地址)

图片

![图片备注](图片地址)

8. 表格（最常用）

| 姓名 | 年龄 | 职业 |
| ---- | ---- | ---- |
| 张三 | 20   | 学生 |

# 对齐格式
| 左对齐 | 居中 | 右对齐 |
| :----- | :---: | -----: |

9. 目录

文档顶部输入，自动生成全文目录

[TOC]

10. 折叠块（笔记神器）

收纳大量内容，默认折叠

<details>
<summary>点击展开</summary>
隐藏的内容、列表、表格
</details>

11. 提示块（高亮备注）

> [!NOTE] 提示
> 普通说明文字

> [!WARNING] 警告
> 注意事项

12. 转义字符

特殊符号前加 \ 取消格式，原样输出：`# * - ``

极简使用总结

日常记笔记只需掌握：标题、列表、表格、代码块、折叠块、引用，足够应对99%场景。
